# incuSens
Project for my studies at Pforzheim University.
PCB senses different environmental Conditions in a biological Lab.

## Hardware
The hardware is designed with [KiCad 6](https://www.kicad.org/). All manufacturing data can be generated through the board file.

## Project Dependencies
This project consists of
- [IncuSens Hardware (this Project)](https://github.com/useless-branch/incusens)
- [IncuLog](https://github.com/useless-branch/inculog)
- [IncuSens Firmware](https://github.com/useless-branch/fw-incusens)
